Shortcuts for writing various arrows:
- →←↑↓
- ▶◀▲▼
- ⇒⇐⇑⇓
- ↔↕⇔⇕⇄⇅
- ↗↘↖↙
- ↪↩↻↺↝↜


__
```
(?:^arrow right$)|(?:^\-\>$)
```
__
```js
return "→";
```
__
-> - → (RIGHT arrow).
        Alternative shortcut: __arrow right__.


__
```
(?:^arrow left$)|(?:^\<\-$)
```
__
```js

```
__
<- - ← (LEFT arrow).
        Alternative shortcut: __arrow left__.
